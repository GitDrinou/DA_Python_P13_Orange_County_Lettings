Description du projet
======================

OC Lettings est une application web développée avec Django.

L'application permet de consulter:

* des annonces de locations
* des profils utilisateurs
* les adresses associées aux logements

Le projet intègre également:

* une journalisation applicative avec Sentry
* une pipeline CI/CD avec GitHub Actions
* une conteneurisation Docker
* une publication d'image sur Docker Hub
* un déploiement sur AWS Elastic Beanstalk
* une documentation hébergée sur Read the Docs
